# Description

This is the repository for Team Switzerland. The problem is "Graphing Fleas" from
http://www-math.mit.edu/~dav/projectsB.pdf.

# Setup

Make sure your computer is running Python 3.

Dependencies
    numpy
    matplotlib

Clone the repo: `git clone https://github.com/wang-daniel/GraphingFleas.git`,

cd into the folder: `cd GraphingFleas`

run the file: `python planegrid.py`.

Then use the right arrow key to simulate steps of the flea.
